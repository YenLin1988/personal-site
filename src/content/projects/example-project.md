## 動機

想用一個靜態網站把自己的學習筆記、作品、想法集中起來。

## 技術選型

- **Astro 5** — 預設靜態輸出，效能極佳。
- **Tailwind CSS** — 樣式快速、好維護。
- **MDX** — 部落格可混用 Markdown 與互動元件。
- **Cloudflare Pages** — 免費、全球 CDN、自動部署。

## 學到的事

- Astro 的 Content Collections 讓 frontmatter 自動 type-check，寫起來很安心。
- Tailwind v4 的設定比 v3 更簡單，幾乎不需要 config 檔。
